
## 🗣️ Food Delivery App 

## 🚀 Technologies Used

- ⚛️ React
- 🌐 React Router DOM
- 🚀 Redux
- 🎨 Tailwind CSS
- 📹 Live Food API

## 🎉 Features

- 📺 Add Item to Cart  
- 💬 Search Food 
- 🎥 Watch Restro Menu Page via click on Restaurant Cart
- 🔍 Search for Cart using the Live Food API and view the results
- 🎥 Browse videos by category using the Live Food API

## 🛠️ Installation and Usage
